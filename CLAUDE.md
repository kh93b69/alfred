# CLAUDE.md

## Инструкции

- Всегда отвечать на русском языке
- Комментарии в коде на русском
- Код понятен Junior разработчику — без наследования, рефлексии, сложных паттернов
- Используй Context7 для документации библиотек перед написанием кода
- Проверяй фронтенд через Playwright после UI-изменений
- Коммиты на русском, деплой — только по запросу пользователя

## Project

Альфред — Telegram-бот "Второй мозг" для автоматизации бизнес-рутины.

Стек: Python 3.11+, aiogram 3.x, Claude API (Anthropic), Railway (хостинг).

Интеграции (поэтапно): Trello, Notion, Whisper (голосовые), python-docx (документы).

## Структура

```
bot/
  main.py          — точка входа, запуск бота
  config.py        — настройки из .env
  handlers/
    start.py       — команда /start
    chat.py        — обработка текстовых сообщений
  services/
    ai.py          — работа с Claude API
knowledge/         — база знаний (текстовые файлы)
```

## Commands

- `python -m bot.main` — запуск бота локально
- `pip install -r requirements.txt` — установка зависимостей
